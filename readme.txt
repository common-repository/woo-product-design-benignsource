== Installation ==

1.    Copy Woo Product Design BenignSource in your plugin directory
2.    In your WordPress administration, go to the Plugins page
3.    Activate the Woo Product Design BenignSource plugin and a subpage for the plugin will appear in your Products.
4.    Go to the Products and make the Woo Product Design for each product
5.    Enjoy!
